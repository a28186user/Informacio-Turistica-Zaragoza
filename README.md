Este sitio web tiene como objetivo promocionar la ciudad de Zaragoza de cara al año 2026, destacando su patrimonio histórico, cultural y gastronómico. La página presenta:

- Información sobre los principales atractivos turísticos
- Sección de lugares imprescindibles
- Calendario de eventos destacados para 2026
- Diseño limpio y profesional que invita a descubrir la ciudad
  
## Características Principales
- **100% Bootstrap** – Todo el diseño está construido únicamente con clases de Bootstrap 5, sin una sola línea de CSS personalizado
- **Diseño responsive** – Se adapta perfectamente a móviles, tablets y escritorio
- **Menú de navegación fijo** – Acceso rápido a todas las secciones desde cualquier punto de la página
- **Hero section con overlay** – Impactante imagen de fondo con gradiente oscuro para mejorar la legibilidad del texto
- **Sistema de tarjetas (cards)** – Presentación visual de los lugares imprescindibles
- **Enlaces de navegación internos** – Desplazamiento suave entre secciones (requiere Bootstrap JS)
